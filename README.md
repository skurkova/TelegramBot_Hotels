# Телеграмм-БОТ компании "HotelsTravel" по подбору отелей.

***Телеграмм-БОТ может подобрать подходящие по цене и расположению отели в любой точке мира.***

## Установка

Для начала необходимо клонировать репозиторий [TelegramBot_Hotels](https://gitlab.skillbox.ru/samira_kurkova/python_basic_diploma/-/tree/homework/TelegramBot_Hotels) 

Установите необходимые библиотеки `pip install -r requirements.txt`

```
pyTelegramBotAPI==4.14.0
python-dotenv==1.0.0
peewee==3.17.0
pydantic==2.5.2
requests==2.31.0
typing-extensions==4.8.0
urllib3==1.26.13
telebot==0.0.5
loguru~=0.7.2
python-telegram-bot-calendar==1.0.5
```

Так же вам понадобится [API сайта с отелями](https://rapidapi.com/apidojo/api/hotels4/)

## Запуск

Запуск Телеграмм-БОТа выполняется командой `main.py` из папки с проектом `TelegramBot_Hotels`.

## Начало работы

Для начала в низу окна нажмите `Начать` или выберете пункт `Меню` ***'/start'***.

При выборе соответствующего пункта `меню` БОТ-помощник подберёт:
- Самые дешёвые отели - ***'/lowprice'***
- Самые дорогие отели - ***'/highprice'***
- Наиболее подходящие отели по расположению от центра города - 
  ***'/customlocation'***

При выборе пункта `меню` ***'/history'***, БОТ-помощник выдаст историю поиска 
отелей.

При выборе пункта `меню` ***'/help'***, БОТ-помощник выводит информационное 
сообщение о возможностях Телеграмм-БОТа.

Если же Вам нужно прервать поиск и перейти в начало, выберете пункт `меню` ***'/reset'***.
